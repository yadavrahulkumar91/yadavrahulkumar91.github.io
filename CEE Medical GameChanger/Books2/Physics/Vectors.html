Vectors and Scalars

Learn the fundamental concepts of units and dimensions, and how to handle errors in measurements.

A. Vectors and Scalars

a. Scalars:

  • All potentials
  • All fluxes
  • Intensity of energy ⚡

b. Vectors

  • Examples

    1. All flux densities
    2. Gradients of all quantities
    3. All field strengths/ intensities
    4. All dipole moments

  • Special types of Vectors

    1. Polar vector
      • Vectors associated with linear directional effect.
      • Displacement (s)
      • Velocity (v)
      • Acceleration (a)
      • Momentum (P)
      • Force (F)
    2. Axial vector
      • Vectors associated with rotation about an axis.
      • Angular displacement (θ)
      • Angular velocity (ω)
      • Angular acceleration (α)
      • Angular momentum (L/ J)
      • Moment of force/ Torque (τ)

c. Tensor

  • Quantity whose magnitude differs from direction to direction
  • Pressure
  • Stress
  • Modulus of elasticity
  • Moment of Inertia
  • Coefficient of viscosity

b. Vector sum of two vectors a⃗ and b⃗.

ii. If a⃗ and b⃗ are inclined at an angle θ and resultant R⃗ makes an angle α with the direction of a⃗, then,

  1. |R⃗| = √(|a⃗|² + |b⃗|² + 2 |a⃗| |b⃗| cosθ)
  2. α = tan⁻¹((b sinθ)/(a + b cosθ)) with a⃗.
  3. β = tan⁻¹((a sinθ)/(b + a cosθ)) with b⃗.

iii. Special cases

  1. If θ = 0°, then |a⃗ + b⃗| = R = a + b (Maximum value).
  2. If θ = 90°, then |a⃗ + b⃗| = R = √(a² + b²).
  3. If θ = 180°, then |a⃗ + b⃗| = (a - b) or (b - a).
  4. When |a⃗| = |b⃗| = A, then
    1. |a⃗ + b⃗| = 2A cos(θ/2)
      1. If θ = 90°, then |a⃗ + b⃗| = √2A.
      2. If θ = 60°, then |a⃗ + b⃗| = √3A.
  5. If |a⃗| = |b⃗| = |a⃗ + b⃗| then, θ = 120°.
  6. If a⃗ + b⃗ = c⃗ and a² + b² = c², then the angle between a⃗ and b⃗ is 90°.
  7. If a⃗ + b⃗ = r⃗ then a⃗, b⃗, and r⃗ lie in the same plane.

  3. β = tan-1((a sin θ)/(b - a cos θ)) with b →.
