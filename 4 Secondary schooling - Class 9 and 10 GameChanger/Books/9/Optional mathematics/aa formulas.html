Trigonometric Formulas in Terms of p, b, and h

  • $\sin(\theta) = \frac{p}{h}$
  • $\cos(\theta) = \frac{b}{h}$
  • $\tan(\theta) = \frac{p}{b}$
  • $\cot(\theta) = \frac{b}{p}$
  • $\sec(\theta) = \frac{h}{b}$
  • $\csc(\theta) = \frac{h}{p}$

Trigonometric Square Formulas

  1. $\sin^2(\theta) + \cos^2(\theta) = 1$
  2. $1 + \tan^2(\theta) = \sec^2(\theta)$
  3. $1 + \cot^2(\theta) = \csc^2(\theta)$
  4. $\sec^2(\theta) - \tan^2(\theta) = 1$
  5. $\csc^2(\theta) - \cot^2(\theta) = 1$
  6. $\tan^2(\theta) + 1 = \sec^2(\theta)$
  7. $\cot^2(\theta) + 1 = \csc^2(\theta)$
  8. $\sin^2(\theta) = \frac{1 - \cos(2\theta)}{2}$
  9. $\cos^2(\theta) = \frac{1 + \cos(2\theta)}{2}$
  10. $\tan^2(\theta) = \sec^2(\theta) - 1$
  11. $\cot^2(\theta) = \csc^2(\theta) - 1$
  12. $\sec^2(\theta) = 1 + \tan^2(\theta)$
  13. $\csc^2(\theta) = 1 + \cot^2(\theta)$

    25. Trigonometric Formulas for (A + B) and (A - B)

    1. $\sin(A + B) = \sin(A)\cos(B) + \cos(A)\sin(B)$
    2. $\cos(A + B) = \cos(A)\cos(B) - \sin(A)\sin(B)$
    3. $\tan(A + B) = \frac{\tan(A) + \tan(B)}{1 - \tan(A)\tan(B)}$
    4. $\sin(A - B) = \sin(A)\cos(B) - \cos(A)\sin(B)$
    5. $\cos(A - B) = \cos(A)\cos(B) + \sin(A)\sin(B)$
    6. $\tan(A - B) = \frac{\tan(A) - \tan(B)}{1 + \tan(A)\tan(B)}$

    Trigonometric Formulas for Expressions of the form sin(A + B) + sin(A - B) and similar

    1. $\sin(A + B) + \sin(A - B) = 2\sin(A)\cos(B)$
    2. $\sin(A + B) - \sin(A - B) = 2\cos(A)\sin(B)$
    3. $\cos(A + B) + \cos(A - B) = 2\cos(A)\cos(B)$
    4. $\cos(A + B) - \cos(A - B) = -2\sin(A)\sin(B)$
    5. $\sin(A + B)\cos(A - B) = \frac{1}{2}[\sin(2A) + \sin(2B)]$
    6. $\cos(A + B)\cos(A - B) = \frac{1}{2}[\cos(2A) + \cos(2B)]$
    7. $\sin(A + B)\sin(A - B) = -\frac{1}{2}[\cos(2A) - \cos(2B)]$

    Trigonometric Formulas for Expressions of the form sin(C) + sin(D) and similar

    1. $\sin(C) + \sin(D) = 2\sin\left(\frac{C + D}{2}\right)\cos\left(\frac{C - D}{2}\right)$
    2. $\sin(C) - \sin(D) = 2\cos\left(\frac{C + D}{2}\right)\sin\left(\frac{C - D}{2}\right)$
    3. $\cos(C) + \cos(D) = 2\cos\left(\frac{C + D}{2}\right)\cos\left(\frac{C - D}{2}\right)$
    4. $\cos(C) - \cos(D) = -2\sin\left(\frac{C + D}{2}\right)\sin\left(\frac{C - D}{2}\right)$
    5. $\sin(C)\cos(D) = \frac{1}{2}[\sin(C + D) + \sin(C - D)]$
    6. $\cos(C)\cos(D) = \frac{1}{2}[\cos(C + D) + \cos(C - D)]$
    7. $\sin(C)\sin(D) = -\frac{1}{2}[\cos(C + D) - \cos(C - D)]$

    Multiple Trigonometric Formulas

    1. $\sin(2\theta) = 2\sin(\theta)\cos(\theta)$
    2. $\cos(2\theta) = \cos^2(\theta) - \sin^2(\theta)$
    3. $\tan(2\theta) = \frac{2\tan(\theta)}{1 - \tan^2(\theta)}$
    4. $\cot(2\theta) = \frac{\cot^2(\theta) - 1}{2\cot(\theta)}$
    5. $\sec(2\theta) = \frac{1}{\cos(2\theta)}$
    6. $\csc(2\theta) = \frac{1}{\sin(2\theta)}$

    Submultiple Trigonometric Formulas

    1. $\sin\left(\frac{\theta}{2}\right) = \pm \sqrt{\frac{1 - \cos(\theta)}{2}}$
    2. $\cos\left(\frac{\theta}{2}\right) = \pm \sqrt{\frac{1 + \cos(\theta)}{2}}$
    3. $\tan\left(\frac{\theta}{2}\right) = \frac{\sin(\theta)}{1 + \cos(\theta)} = \frac{1 - \cos(\theta)}{\sin(\theta)}$
    4. $\cot\left(\frac{\theta}{2}\right) = \frac{\sin(\theta)}{1 - \cos(\theta)} = \frac{1 + \cos(\theta)}{\sin(\theta)}$
    5. $\sec\left(\frac{\theta}{2}\right) = \frac{1}{\cos\left(\frac{\theta}{2}\right)} = \frac{2}{1 + \cos(\theta)}$
    6. $\csc\left(\frac{\theta}{2}\right) = \frac{1}{\sin\left(\frac{\theta}{2}\right)} = \frac{2}{1 - \cos(\theta)}$
    7. $\sin(\theta) = 2\sin\left(\frac{\theta}{2}\right)\cos\left(\frac{\theta}{2}\right)$
    8. $\cos(\theta) = \cos^2\left(\frac{\theta}{2}\right) - \sin^2\left(\frac{\theta}{2}\right)$
    9. $\tan(\theta) = \frac{2\tan\left(\frac{\theta}{2}\right)}{1 - \tan^2\left(\frac{\theta}{2}\right)}$
    10. $\cot(\theta) = \frac{\cot^2\left(\frac{\theta}{2}\right) - 1}{2\cot\left(\frac{\theta}{2}\right)}$
    11. $\sec(\theta) = \frac{1}{\cos(\theta)} = \frac{2}{1 + \cos(\theta)}$
    12. $\csc(\theta) = \frac{1}{\sin(\theta)} = \frac{2}{1 - \cos(\theta)}$

    Derivative Formulas for Common Functions

    • $\frac{d}{dx}(k) = 0$ (where $k$ is a constant)
    • $\frac{d}{dx}(x^n) = nx^{n-1}$ (power rule)
    • $\frac{d}{dx}(e^x) = e^x$ (derivative of exponential function)
    • $\frac{d}{dx}(\ln(x)) = \frac{1}{x}$ (derivative of natural logarithm)
    • $\frac{d}{dx}(\log_b(x)) = \frac{1}{x \ln(b)}$ (derivative of logarithm base $b$)
    • $\frac{d}{dx}(\sqrt{x}) = \frac{1}{2\sqrt{x}}$ (derivative of square root)

    Derivative Formulas for Hyperbolic Functions

    • $\frac{d}{dx}(\sinh(x)) = \cosh(x)$
    • $\frac{d}{dx}(\cosh(x)) = \sinh(x)$
    • $\frac{d}{dx}(\tanh(x)) = \sech^2(x)$
    • $\frac{d}{dx}(\coth(x)) = -\csch^2(x)$
    • $\frac{d}{dx}(\sech(x)) = -\sech(x) \tanh(x)$
    • $\frac{d}{dx}(\csch(x)) = -\csch(x) \coth(x)$

    Derivatives Formulas for Trigonometric Functions

    • $\frac{d}{dx}(\sin(x)) = \cos(x)$
    • $\frac{d}{dx}(\cos(x)) = -\sin(x)$
    • $\frac{d}{dx}(\tan(x)) = \sec^2(x)$
    • $\frac{d}{dx}(\cot(x)) = -\csc^2(x)$
    • $\frac{d}{dx}(\sec(x)) = \sec(x)\tan(x)$
    • $\frac{d}{dx}(\csc(x)) = -\csc(x)\cot(x)$

    Derivatives Formulas for Inverse Trigonometric Functions

    • $\frac{d}{dx}(\arcsin(x)) = \frac{1}{\sqrt{1-x^2}}$
    • $\frac{d}{dx}(\arccos(x)) = -\frac{1}{\sqrt{1-x^2}}$
    • $\frac{d}{dx}(\arctan(x)) = \frac{1}{1+x^2}$
    • $\frac{d}{dx}(\arccot(x)) = -\frac{1}{1+x^2}$
    • $\frac{d}{dx}(\arcsec(x)) = \frac{1}{x\sqrt{x^2-1}}$
    • $\frac{d}{dx}(\arccsc(x)) = -\frac{1}{x\sqrt{x^2-1}}$

    Antiderivative Formulas for Common Functions

    • $\int k \, dx = kx + C$ (where $k$ is a constant)
    • $\int x^n \, dx = \frac{1}{n+1}x^{n+1} + C$ (power rule)
    • $\int e^x \, dx = e^x + C$ (antiderivative of exponential function)
    • $\int \frac{1}{x} \, dx = \ln|x| + C$ (antiderivative of inverse function)
    • $\int \frac{1}{x\ln(b)} \, dx = \log_b|x| + C$ (antiderivative of logarithm base $b$)
    • $\int \frac{1}{2\sqrt{x}} \, dx = \sqrt{x} + C$ (antiderivative of square root)

    Antiderivative Formulas for Trigonometric Functions

    • $\int \sin(x) \, dx = -\cos(x) + C$
    • $\int \cos(x) \, dx = \sin(x) + C$
    • $\int \tan(x) \, dx = -\ln|\cos(x)| + C$
    • $\int \cot(x) \, dx = \ln|\sin(x)| + C$
    • $\int \sec(x) \, dx = \ln|\sec(x) + \tan(x)| + C$
    • $\int \csc(x) \, dx = \ln|\csc(x) - \cot(x)| + C$

    Antiderivative Formulas for Hyperbolic Functions

    • $\int \sinh(x) \, dx = \cosh(x) + C$
    • $\int \cosh(x) \, dx = \sinh(x) + C$
    • $\int \tanh(x) \, dx = \ln|\cosh(x)| + C$
    • $\int \coth(x) \, dx = \ln|\sinh(x)| + C$
    • $\int \sech(x) \, dx = \arctan(e^x) + C$
