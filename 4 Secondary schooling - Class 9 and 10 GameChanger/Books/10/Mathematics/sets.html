Formulas and Notations in Set Theory

  • Set Notation: $A$, $B$, $C$, ... (capital letters for sets)
  • Element: $x \in A$ (element $x$ belongs to set $A$)
  • Subset: $A \subseteq B$ (set $A$ is a subset of set $B$)
  • Proper Subset: $A \subset B$ (set $A$ is a proper subset of set $B$)
  • Union: $A \cup B$ (union of sets $A$ and $B$)
  • Intersection: $A \cap B$ (intersection of sets $A$ and $B$)
  • Complement: $A^c$ or $\overline{A}$ (complement of set $A$)
  • Empty Set: $\emptyset$ or $\{\}$ (symbol for the empty set)
  • Universal Set: $U$ (set containing all elements under consideration)
  • Set Difference: $A \setminus B$ or $A - B$ (elements in $A$ but not in $B$)
  • Cartesian Product: $A \times B$ (cartesian product of sets $A$ and $B$)
  • Power Set: $\mathcal{P}(A)$ (set of all subsets of set $A$)
  • Cardinality: $|A|$ (number of elements in set $A$)
  • Set Equality: $A = B$ (sets $A$ and $B$ are equal)

Formulas and Identities for Set Operations

  • Union: $A \cup B = \{x \mid x \in A \text{ or } x \in B\}$
  • Intersection: $A \cap B = \{x \mid x \in A \text{ and } x \in B\}$
  • Complement: $A^c = \{x \mid x \notin A\}$
  • Set Difference: $A \setminus B = \{x \mid x \in A \text{ and } x \notin B\}$
  • Symmetric Difference: $A \Delta B = (A \cup B) \setminus (A \cap B)$
  • De Morgan's Laws:
    • $\overline{A \cup B} = \overline{A} \cap \overline{B}$
    • $\overline{A \cap B} = \overline{A} \cup \overline{B}$
  • Distributive Laws:
    • $A \cup (B \cap C) = (A \cup B) \cap (A \cup C)$
    • $A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$
  • Set Identity: $A \cup \emptyset = A$
  • Set Identity: $A \cap U = A$ (where $U$ is the universal set)
  • Set Complement: $A \cup A^c = U$
  • Inclusion-Exclusion Principle: $|A \cup B| = |A| + |B| - |A \cap B|$
