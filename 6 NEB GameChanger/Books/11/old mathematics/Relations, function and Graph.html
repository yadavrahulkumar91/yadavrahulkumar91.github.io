The function \(f: \mathbb{R} \rightarrow \mathbb{R}\) is \(f(x) = \frac{1}{1+e^{-x}}\).
To show: \(f\) is one-to-one and onto.
a. \(f\) is one-to-one:
Let \(x, y \in \mathbb{R}\) such that \(x \neq y\). Then to show \(f(x) \neq f(y)\).
Since \(x \neq y\),
\[e^{-x} \neq e^{-y}\]
\[1+e^{-x} \neq 1+e^{-y}\]
\[\frac{1}{1+e^{-x}} \neq \frac{1}{1+e^{-y}}\] (since \(f(x) = \frac{1}{1+e^{-x}}\))
Therefore, \(f\) is one-to-one.
b. \(f\) is onto:
Let \(y \in \mathbb{R}\). We need to find an \(x \in \mathbb{R}\) such that \(f(x) = y\).
Given \(f(x) = \frac{1}{1+e^{-x}} = y\).
Solving for \(x\),
\[\frac{1}{y} = 1+e^{-x}\]
\[e^{-x} = \frac{1}{y} - 1\]
\[e^{-x} = \frac{1-y}{y}\]
\[-x = \ln\left(\frac{1-y}{y}\right)\]
\[x = -\ln\left(\frac{1-y}{y}\right)\]
Since \(-\ln\left(\frac{1-y}{y}\right) \in \mathbb{R}\), we can find \(x\) for any given \(y\).
Hence, \(f\) is onto.
Therefore, the function is both one-to-one and onto.
