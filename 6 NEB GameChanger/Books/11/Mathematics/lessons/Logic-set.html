\(A - B^c = \{x : x \in A \text{ and } x \notin B^c\}\)

\(= \{x : x \in A \text{ and } x \in (A \cap B)\}\)

\(= \{x : x \in A \text{ and } x \in B\} = \{x : x \in (A \cap B)\} = A \cap B\)
