Soln:

a. \(A \cup B = \{1, 2, 3, 4\} \cup \{3, 5, 7, 8\} = \{1, 2, 3, 4, 5, 7, 8\}\)

b. \(A \cap C = \{1, 2, 3, 4\} \cap \{1, 2, 7, 8\} = \{1, 2\}\)

c. \((A - B) \cap C = (\{1, 2, 3, 4\} - \{3, 5, 7, 8\}) \cap \{1, 2, 7, 8\} = \{1, 2, 4\} \cap \{1, 2, 7, 8\} = \{1, 2\}\)

d. \(A \cup C = \{1, 2, 3, 4\} \cup \{1, 2, 7, 8\} = \{1, 2, 3, 4, 7, 8\}\)

So, \(\overline{A \cup C} = U - (A \cup C)\)

\(\overline{A \cup C} = \{5, 6, 9, 10\}\)

e. \(\overline{B} = U - B = \{1, 2, 3, \dots, 9, 10\} - \{3, 5, 7, 8\} = \{1, 2, 4, 6, 9, 10\}\)
